# How to contribute a new feature to deer-flow

_Create a new branch, implement the feature, and submit a pull request_

## Steps

1. Create a new branch using `git checkout -b feature/new-feature`
2. Implement the new feature and add relevant tests
3. Commit the changes using `git add .` and `git commit -m 'Added new feature'`
4. Push the branch to the remote repository using `git push origin feature/new-feature`
5. Submit a pull request to the main branch using the GitHub web interface

## Pitfalls

- ⚠️ Not following the deer-flow coding standards and conventions
